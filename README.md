# Philippine Military Academy (PMA) DTR Processing System

**A web-based application that automates the management of employee Daily Time Records (DTR).**

The application is divided into two main modules:

- Admin Module – for managing and processing Daily Time Records (DTR) for employees
- Employee Module – for viewing and printing individual DTR record
